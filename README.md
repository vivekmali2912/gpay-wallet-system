# 💳 GPay Wallet System (Python Project)

A simple console-based Digital Wallet and UPI Payment simulation system built using Python and Object-Oriented Programming concepts.

---

## 📌 Features

- User Registration
- Merchant Registration
- Add Money to Wallet
- Send Money via UPI PIN
- Wallet Balance Check
- Transaction History
- Secure PIN Verification
- OOP Based Architecture

---

## 🏗 Technologies Used

- Python 3
- OOP Concepts
- datetime module

---

## 📂 Project Structure

```
main.py
README.md
requirements.txt
```

---

## ▶️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/your-username/gpay-wallet-system.git
```

2. Open project folder

```
cd gpay-wallet-system
```

3. Run the program

```
python main.py
```
